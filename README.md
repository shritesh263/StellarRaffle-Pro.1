# Stellar Raffle

## Features
- User authentication
- Ticket purchasing
- Raffle drawing

## Tech Stack
- Frontend: React
- Backend: Node.js
- Database: MongoDB

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/shritesh263/StellarRaffle-Pro.1.git
   ```
2. Navigate to the project directory:
   ```bash
   cd StellarRaffle-Pro.1
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
